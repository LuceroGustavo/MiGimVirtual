# MiGymVirtual

App de gestión de gimnasio virtual: ejercicios, series, rutinas y hoja por token. Panel único de profesor (evolución desde Mattfuncional).

**Documentación:** En la carpeta **`Documentacion/`**:

1. **`Documentacion/LEEME_PRIMERO.md`** – Contexto del proyecto, acceso y dónde está cada cosa en el código.
2. **`Documentacion/PLAN_DE_DESARROLLO_MiGymVirtual.md`** – Plan de desarrollo (BD, renombre, eliminaciones, responsive).
3. Según necesidad: AYUDA_MEMORIA.md, DOCUMENTACION_UNIFICADA.md.
4. **Backups y despliegue Ubuntu:** `DOCUMENTACION_UNIFICADA.md` §2 y `Documentacion/servidor/DESPLIEGUE-SERVIDOR.md` §6.6 (carpeta `backup`: se crea sola al primer guardado; opcional ruta absoluta `migimvirtual.backups.dir`).

**Stack:** Spring Boot, Thymeleaf, MySQL, Bootstrap. Código en `src/main/java/com/MiGymVirtual/`.
